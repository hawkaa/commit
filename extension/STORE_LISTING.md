## Detailed Description

Commit surfaces trust signals you can verify — right where you make decisions.

When evaluating a GitHub repo, a business, or a service, Commit shows a Trust Score (0–100) based on publicly verifiable data. No opinions. No ratings. Just evidence.

Phase 1 covers GitHub repositories in your browser and Google search results. More entity types and ZK-verified endorsements are coming.

How it works:
• Visit a GitHub repo → see its Commit Score inline
• Search Google → scores appear next to GitHub links
• Click any score → full trust card with signal breakdown

What makes it different:
• All signals are from public, verifiable sources
• No account required. No personal data collected.
• Open source: https://github.com/hawkaa/commit

Commit Score is a trust primitive — one number that means something.
